Live public cameras, one tap away

Holes is an open-source Flutter app for Android that browses a bundled catalog of ~4,700 HTTPS live streams — traffic cams, city views, outdoor feeds. Search, filter, star favorites, skip bad feeds. No map. No account.

Features

Search & filter

By name, city, country, tags. Country, category, HLS/MJPEG, favorites-only.

Random browse

Shuffle through feeds. Remember last camera. Skip & hide broken streams.

HLS playback

Real video on Android. Health badge, fullscreen, open in browser fallback.

Privacy-first

No account. Favorites and prefs stay on your device. Public streams only.

Onboarding

Choose stream types and block categories on first launch.

Blocklist

Bundled URL blocklist and per-user skip/hide for bad feeds.

Install (Android)

  1. Download app-release.apk from GitHub Releases.
  2. Verify the SHA256 checksum (see below) before installing.
  3. On your phone: allow install from unknown sources / USB (MIUI: Install via USB).
  4. Open the APK and install. No Google Play required.

What is this site? This GitHub Pages page is the project homepage — it does not run the app in your browser. Holes is an Android app. Build from source: see instructions.md.
